## Partisjonering av noder i en graf i sykler og stier 

Det er ofte avhengigheter rundt om i nettverk. Man kan vise disse som en graf hvor avhengighetene er kanter mellom noder. I mitt prosjekt vil jeg ta for meg å undersøke hvordan man kan finne sykler i avhengighetene eller stier som kan løses opp. Et enkelt eksempel er hvor en node a er avhengig av en node b, men b er avhengig av a. Dette er en sykel og kan løses opp og nodene som blir løst er {a,b}. Jeg vil undersøke algoritmer som kan finne det største settet av noder som kan bli løst opp.
